# RecipeRAG
A knowledge RAG system for your recipes.

THIS IS A WIP!!

You can initialise the database with initialise_database.py
This requires having non empty csv files

You can insert data straight into the Ladybugdb using main.py

You can save the data from the ladybug db to the csv just as a safety measure, you can treat it like a backup since then initialise_database.py can convert the csv back into the db you want.
